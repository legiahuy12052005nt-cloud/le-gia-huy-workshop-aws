### Week 2 Outcomes and Deliverables:

* Networking: I successfully configured an operational network boundary using Amazon Virtual Private Cloud (Amazon VPC), splitting custom IP spaces, and attaching an Internet Gateway.
* Compute Server: Understood the mechanics of running virtual instances via Amazon Compute Cloud (Amazon EC2), deploying test packages, and handling firewall traffic with Security Groups.
* Application Security: Implemented IAM Role (AWS IAM) integrations attached directly to EC2 nodes, allowing background apps to query AWS backends safely without storing long-term credentials in plain code text.
* Development IDE: Got comfortable using the AWS Cloud9 console tool to compile scripts and run CLI parameters directly within a browser frame without resource dependencies on local machines.
* Static Hosting: Mastered the basic attributes of Amazon S3 buckets, completing a functional deployment of a public website setup with high file performance and optimized runtime cost metrics.
