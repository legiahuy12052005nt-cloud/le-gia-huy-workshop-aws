### Week 4 Achievements:

* Mastered and successfully implemented tagging strategies to orchestrate systematic resource grouping and allocation models via resource groups, significantly optimizing large-scale governance operations.
* Engineered secure iam policy definitions anchored to specific resource tag metadata, strictly limiting and isolating access vectors into active ec2 compute assets under strict least privilege access controls.
* Successfully eliminated perimeter security vulnerabilities associated with exposed open SSH management ports by deploying aws system manager and session manager to establish secure, encrypted remote terminal sessions.
* Deployed a robust, multi-tier performance monitoring and automated incident alerting layout:
  * Captured system metrics and performance trends using customized logs and metric structures within amazon cloudwatch, routing the streams into centralized analytics dashboards powered by grafana.
  * Extinguished unnecessary infrastructure operational spend by deploying event-driven serverless code blocks on aws lambda, which continuously handles idle compute shutdowns and reports real-time power states back to technical slack channels.
* Acquired an advanced understanding of strategic enterprise migration frameworks and operational procedures:
  * Understood the lifecycle of virtual machine disk conversions from on-premises hypervisors into deployable cloud assets using the aws vm import/export engine.
  * Acquired proficiency in utilizing sct to evaluate and refactor legacy database schema code bases, working alongside live replication pipelines in dms to synchronize active transactional databases with zero downtime.
* Mastered declarative resource assembly syntax, successfully authoring template parameters using aws cloudformation to easily instantiate, modify, or terminate entire software stacks in a repeatable and predictable manner.
